# 花点萌

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test

## 文件夹介绍
- 路由页面在pages里面, router.js会自动查找pages里面的文件,动态生成前端路由。
- components 里面全部为组件
- commons 里面主要存放js文件
- assets 里面主要存放静态资源和api.js
- store vuex
